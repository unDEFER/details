# @Grep

Команда `grep` помогает нам найти все описанные или используемые детали.
Вот список опций которые мы используем для этого:

-E -- использовать расширенные регулярные выражения

-I -- обрабатывать только текстовые файлы

-h -- не выводить имена файлов

-r -- искать файлы рекурсивно

-o -- вывести только совпадение, а не всю строку
